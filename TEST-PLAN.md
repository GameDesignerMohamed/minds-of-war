## Priority 1: Critical System Tests (NEW)

### AttackSystem

- Unit attacks target, damage applied correctly (uses DamageFormula)
- Target at 0 HP emits UNIT_DIED event
- Attack cooldown prevents rapid fire
- Attack range validation (melee vs ranged)
- Dead unit stops attacking

### ConstructionSystem

- Building progresses each tick (HP scales with progress)
- Building completes at 100%, emits BUILDING_COMPLETE
- Incomplete building has minimum 1 HP
- Worker assignment tracking

### TrainingQueueSystem

- Unit queued, trains for correct duration, spawns entity
- Multiple units in queue train sequentially
- Missing unit definition throws error (not silent fallback)
- Supply cost checked before spawn

### SpellSystem

- Heal spell restores HP (capped at max)
- AoE spell damages all enemies in radius
- Chain lightning bounces to nearest unhit enemy
- Buff spell applies BuffEntry with correct duration
- Cooldown prevents re-cast

### Victory Condition

- Human Keep destroyed → lose screen
- Orc Stronghold destroyed → win screen

## Priority 2: Infrastructure Tests (NEW)

### GameLoop

- Fixed timestep: N ticks per accumulated time
- Spiral-of-death: large delta clamped to maxFrameTime
- Alpha interpolation between 0 and 1
- start()/stop() lifecycle
- No ticks when stopped

### EventBus

- on() + emit() delivers payload
- off() / unsubscribe removes listener
- once() fires exactly once then auto-removes
- Snapshot: listener added during emit doesn't fire in same cycle
- clearAll() removes all listeners
- listenerCount() accurate after add/remove

## Priority 3: Edge Cases to Verify

### Economy

- spend() with insufficient resources returns false (after guard added)
- Resources never go negative
- Simultaneous spend in same tick (AI + player)

### Pathfinding

- No valid path → movement target cleared, no crash
- Start tile blocked → returns null gracefully
- Path through fog of war

### AI

- AI with no buildings → doesn't crash
- AI with no resources → stays in EcoState
- DefenseOverride triggers on attack detection
